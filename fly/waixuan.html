3、十大世界语言 💬 World Language

　　世界语言是指在世界上广泛使用的语言，不局限于一国一洲。受益于互联网的全球连接，世界语言在云媒体上得到更广泛的传播。　　中英法，西阿俄，德日韩葡，这是大萌总结的世界语言排行榜，列举如下： 
　　中文 = zh📍 = Chinese = zhongwen ，世界母语人口(超9亿*)最多的语言，也作：汉语；
　　英文 = en🍴 = English　　　　　　　，世界使用国家最多(超60个)的语言；
　　法文 = fr = Francais = Français ，世界使用国家(超45个)第二多的语言；
西班牙文 = es = Spanish = Español 　，世界母语人口(近5亿)第二多的语言；
阿拉伯文 = ar = Arab　 =　عربى　　，世界使用国家(超20个)第三多的语言，*也作：لعربية ；
　　俄文 = ru = Russian = русский ，欧洲母语人口最多的语言，也作：ру́сский язы́к；
　　德文 = de = German = Deutsch　，欧盟内使用最广泛的母语(约0.9亿)；
　　日文 = ja = Japenese = 日本語　，约有1.2亿人使用，也作：にほんご；
　　韩文 = ko = Korean　 = 조선말　，约有0.8亿人使用，也作：한국어，朝鲜语；
葡萄牙文 = pt = Portuguese = Português，世界母语人口(超2亿)第六多的语言。
　　本排行榜综合考虑世界语言文字的三个方面：母语人口、使用国家、媒体影响力。本榜不包括印度半岛的印地语和孟加拉语，二者使用人数约为3亿和2亿，但媒体影响力较小，在中国对外宣介媒体中几乎见不到这两种文字。列表中的2字母缩写是 ISO 639-1 的标准代码。
　　*注1：中文的母语人口数字偏保守，可能排除了中文繁体或粤语方言，但从‘书同文’的角度，母语为中文的人口应超过12亿，世界第一没有争议。
　　*注2：阿拉伯文字与其他几大语言有个最大的不同，其书写方向从右到左，在网页上选择阿拉伯文本的时候，请注意鼠标指针移动方向。

4、中国人民解放军 PLA

　　中国人民解放军，简称‘解放军’，是中华人民共和国的主要武装力量。　解放军诞生于1927年8月1日的南昌起义，先后使用过：国民革命军第二方面军、中国工农革命军、中国工农红军、八路军、新四军 等名称，1947年10月10日，毛泽东起草的《中国人民解放军宣言》里正式启用现名。　解放军在中国共产党历史与中华人民共和国历史上发挥了重要作用，参加了土地革命战争、抗日战争、解放战争、朝鲜战争、中印战争、中苏边境冲突、中越战争等等，从一个胜利走向另一个胜利。每年的八月一日为建军节。
　　解放军由中国共产党建立，由中央军事委员会领导，中共中央军委主席统率全军。　解放军现役部队由陆军、海军、空军、火箭军、战略支援部队五大军种以及中央军委联勤保障部队组成，现役总人数约200万人。　自2020年起，解放军实行一年两次征兵、两次退役。征兵工作由国防部征兵办公室组织实行，详情见全国征兵网。
　　中国人民解放军的英文为：People's Liberation Army of China，英文缩写：PLA。

4.1　解放军为什么是人民的军队

　　1927年在江西永新县三湾村，毛泽东领导了举世闻名的“三湾改编”，确立了：党支部建在连上、设立士兵委员会，部队中出现了官兵一致、上下平等的新型官兵关系。　
　　2018年颁布的《中国人民解放军内务条令》第四条规定：“紧紧地和人民站在一起，全心全意地为人民服务，是这支军队的唯一宗旨。中国人民解放军必须始终不渝地保持人民军队的性质，忠于党，忠于社会主义，忠于祖国，忠于人民。中国人民解放军的任务是，巩固国防，抵抗侵略，保卫祖国，保卫人民的和平劳动，参加国家建设事业。”
　　2017年在庆祝中国人民解放军建军90周年大会上，习主席所说，人民军队始终和人民同呼吸、共命运、心连心，完全彻底为人民奋斗，哪里有敌人，哪里有危难，哪里就有人民子弟兵。

4.2　解放军有关部门的网站

　　解放军出于军事保密需要，大部分部门网站不公开。　全军唯一新闻门户网站是‘中国军网’，使用中国自己的国别顶级域‘.cn’和中国金融认证中心的数字安全证书。　如果发生互联网战争，.com容易被中断，而国内访问.cn可以保持解析。本页面收录的解放军官方微博大部分采用国别域名 weibo.cn，而不是常见的通用域名 weibo.com。
　　全军共有军事院校44所，其中面向社会高考招生的有27所(2022年)，大部分不公开网站，个别院校开设微博，请见本页第一段。　在中央军委直属的三所军事院校中：军事科学院、国防大学、国防科技大学，只有最后一所国防科大公开网站。
