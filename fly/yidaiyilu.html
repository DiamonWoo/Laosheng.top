3、云媒体的世界语言 💬 World Language

　　世界语言是指在世界上广泛使用的语言，不局限于一国一洲。受益于互联网的全球连接，世界语言在云媒体上得到更广泛的传播。想观看一带一路的云媒体，首先要认识这些世界语言。 　　中英西，法阿俄，德日韩葡，这是大萌对云媒体总结的世界语言排行榜，列举如下，其中2字母的缩写是 ISO 639-1 的标准。

3.1 十大语言排行榜

　　中文 = zh = Chinese = zhongwen ，世界母语人口(超9亿*)最多的语言，也作：汉语；
　　英文 = en = English　　　　　　　，世界使用国家最多(超60个)的语言；
　　法文 = fr = Francais = Français ，世界使用国家(超45个)第二多的语言；
西班牙文 = es = Spanish = Español 　，世界母语人口(近5亿)第二多的语言；
阿拉伯文 = ar = Arab　 =　عربى　　，世界使用国家(超20个)第三多的语言，*也作：لعربية ；
　　俄文 = ru = Russian = русский ，欧洲母语人口最多的语言，也作：ру́сский язы́к；
　　德文 = de = German = Deutsch　，欧盟内使用最广泛的母语(约0.9亿)；
　　日文 = ja = Japenese = 日本語　，约有1.2亿人使用，也作：にほんご；
　　韩文 = ko = Korean　 = 조선말　，约有0.8亿人使用，也作：한국어，朝鲜语；
葡萄牙文 = pt = Portuguese = Português，世界母语人口(超2亿)第六多的语言。
　　本排行榜综合考虑世界语言文字的三个方面：母语人口、使用国家、媒体影响力。本榜不包括印度半岛的印地语和孟加拉语，二者使用人数约为3亿和2亿，但媒体影响力较小，在中国对外宣介媒体中几乎见不到这两种文字。
　　*注：中文的母语人口数字偏保守，可能排除了中文繁体或粤语方言，但从‘书同文’的角度，母语为中文的人口应超过12亿，世界第一没有争议。
　　*注：阿拉伯文字与其他几大语言有个最大的不同，其书写方向和其他文字相反，是从右到左，在网页上选择阿拉伯文本的时候，请注意鼠标指针移动方向。

4.1 为什么收录通讯社？

　　通讯社是采集和销售新闻的机构，也叫新闻社，英文为 News Agencies 或 News Service。
　　为了更多的采集，通讯社 通常在多个地方设立记者站；为了更多的销售，通讯社 通常会提供新闻的多个语种版本（大萌，2022）。在理性的情况下，为了使新闻让更多地区的人群接受，通讯社的新闻稿件一般会避免明显的倾向性，保持中立与客观。所以，收录世界各国的通讯社网站，是一个了解世界的较好途径。（可惜有时候，世界是疯狂的）。
　　不是所有的新闻社都在网站免费提供新闻，他们主要面对集体客户和付费客户。好消息是，向最终用户免费提供新闻的通讯社网站已经足够多，一带一路页面重点收录这一类境外通讯社。按覆盖地区计算的世界六大通讯社均在此类：法新社 AFP、美联社 AP、(英国)路透社 Reuters、(西班牙)埃菲社 EFE、新华社 XinHua、(俄罗斯)塔斯社 Tacc。
